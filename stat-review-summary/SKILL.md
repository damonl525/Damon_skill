# Statistical Review Summary

作为临床统计师，系统性总结参考研究资料的标准化模板。

## Overview

This skill helps you:
- 创建标准化的临床研究总结文档
- 系统性整理试验设计、统计方法和结果
- 记录监管机构的关注点和态度
- 分析小样本量等特殊统计方法
- 为药物开发决策提供结构化参考

## 使用场景

### 1. 文献回顾总结
- 总结已发表的临床试验论文
- 整理多个研究的关键发现
- 比不同研究的统计方法

### 2. 监管文件准备
- 准备IND/NDA申报材料中的统计部分
- 总结FDA审评报告的关键要点
- 整理咨询委员会的讨论要点

### 3. 内部决策支持
- 为项目团队提供研究概览
- 支持药物开发策略讨论
- 评估统计方法的适当性

## 模板结构

### 核心章节

| 章节 | 内容要点 | 重要性 |
|------|----------|--------|
| **上市依据的基础** | 核心试验、支持性试验、药物/适应症 | ⭐⭐⭐⭐⭐ |
| **试验设计** | 设计类型、样本量、随机化、治疗、盲法 | ⭐⭐⭐⭐⭐ |
| **主要终点** | 终点名称、临床意义、评分逻辑 | ⭐⭐⭐⭐⭐ |
| **统计分析方法** | 主要分析、多重性控制、缺失数据处理 | ⭐⭐⭐⭐⭐ |
| **主要结果** | 关键结果数据、置信区间、p值 | ⭐⭐⭐⭐⭐ |
| **监管看法** | 咨询委员会投票、监管关注点、审稿人评论 | ⭐⭐⭐⭐ |
| **值得关注的统计方法** | 小样本方法、贝叶斯方法、自适应设计 | ⭐⭐⭐ |

## 使用方法

### 基本使用

1. **准备研究资料**
   - 临床试验方案（Protocol）
   - 统计分析计划（SAP）
   - 临床研究报告（CSR）
   - FDA审评报告（StatR, MedR）
   - 咨询委员会记录

2. **使用模板填写**
   - 按照模板结构提取关键信息
   - 保持客观、准确的描述
   - 注明数据来源和页码

3. **审核和完善**
   - 检查数据一致性
   - 确保关键信息完整
   - 请同事审阅

### 模板文件

- `template.md` - 空白模板，可直接填写
- `example.md` - 完整示例（基于LUXTURNA案例）

## 最佳实践

### 1. 信息提取

**试验设计部分**
- 明确区分主要试验和支持性试验
- 详细描述随机化方法和分层因素
- 说明盲法实施的具体情况

**统计方法部分**
- 记录主要分析和敏感性分析
- 说明多重性控制策略
- 描述缺失数据处理方法

**结果部分**
- 报告效应量和置信区间
- 包含关键的亚组分析结果
- 注意结果的一致性和稳健性

### 2. 监管视角

**关注监管机构的关切**
- 记录咨询委员会的讨论要点
- 注意审稿人提出的问题
- 评估监管决策的依据

**风险获益评估**
- 总结监管机构对获益风险的看法
- 记录任何安全性担忧
- 注意标签相关的讨论

### 3. 特殊方法记录

**小样本量方法**
- 记录样本量计算依据
- 描述使用的统计方法
- 评估方法的适当性

**贝叶斯方法**
- 记录先验分布的选择
- 描述后验推断方法
- 评估与频率方法的比较

**自适应设计**
- 描述适应性规则
- 记录中期分析计划
- 评估对I类错误的影响
